MUSICAL PAINT
university project
university of the arts london
year 1 / block 1
2021

play and experiment in p5. js which is a java script client-side library for creating graphic and interactive experiences, based on the core principles of processing.


---------------------------------
i wanted to create a musical painting tool in which you add notes, looping or single ones and each note produces rectangles or ellipses (you decide). the size of the shapes depends on the volume of the notes and the volume of the noises you make. you can choose the shapes to be filled or not.
